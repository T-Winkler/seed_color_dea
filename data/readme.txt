## Data

All data generated from the gene expression quantification and differential expression analysis.

Folders:

sequencing_adapters:
sequencing adapters used for adapter trimming

featurecounts:
output of gene expression quantification using featurecounts 

differential_expression_analysis:
output files from differential expression analysis using DESeq2

import:
annotation files of betalain and flavonoid pathway genes and MYB transcription factors

functional_annotation:
output from online version of eggNOG mapper, used for import into R for GO-term analysis of differentially expressed genes
